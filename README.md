# feedrr
feedrr - RSS/News Aggregator (mvp)

## Project Scope

Priority of:

- 2x RSS sources
- Mobile/Web view similar to (iOS app `feeed`)
- Modern language (Python/Go)
- Github pages hosted (similar to osmos:feed ir FreshRss)
- Context/topic tagging (using lightweight llm) 
- Content deduplication (using lightweight llm)
- Bionic reader mode
- Local db/records with rollback or versioning
